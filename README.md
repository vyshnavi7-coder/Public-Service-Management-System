# Public Service Complaint Management System

A Django-based web application for submitting and tracking public service complaints.

## Features
- User Registration & Login
- Complaint Submission
- Complaint Tracking
- Admin Dashboard
- Responsive UI

## Tech Stack
- Python
- Django
- HTML
- CSS
- Bootstrap
- SQLite
- GitHub
- Render

## Live Demo
🔗 Public Website: https://public-service-management-system.onrender.com

## Installation

```bash
git clone https://github.com/vyshnavi7-coder/Public-Service-Management-System.git
cd Public-Service-Management-System

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
